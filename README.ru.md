# Плагин Tuya

Плагин интеграции устройств экосистемы Tuya в систему умного дома osysHome.

## Возможности

- **Облачное управление**: Использует Tuya IoT Platform API с MQTT push-уведомлениями
- **Локальное управление**: Прямое управление по локальной сети через tinytuya (без задержек облака)
- **Автообнаружение**: Автоматическое обнаружение устройств из аккаунта Tuya
- **Ручное добавление**: Добавление устройств вручную с device ID и local key
- **Универсальная поддержка**: Поддерживает выключатели, освещение, датчики, климат, шторы и др.

## Настройка

1. Создайте Cloud Project на [iot.tuya.com](https://iot.tuya.com)
2. Получите Access ID и Access Secret
3. Привяжите аккаунт приложения Smart Life/Tuya Smart
4. Настройте плагин в панели администратора

## Поддерживаемые типы устройств

- Выключатели и розетки
- Умное освещение (RGB, диммеры)
- Датчики (температура, влажность, движение, дверь)
- Климатические устройства (термостаты, кондиционеры)
- Шторы и жалюзи
- Все прочие устройства через универсальный маппинг DPS
